ComputerVision

In de eerste week van HCI gaat het over computer vision. Computer vision is, bijvoorbeeld door menselijke input via een camera objecten kan besturen.

Workshop

Assignment 1

Dit was de eerste opdracht die ik heb gemaakt tijdens de workshop van hci. Zoals de bovenstaande afbeelding laat zien werd er van ons verwacht het zo goed mogelijk na te maken van het voorbeeld. Ik had nog nooit eerder met dit programma gewerkt. In de void setup konden de verschillende vromen onder elkaar worden gezet met de daarbij behorende specificaties. Fill geeft de vulling van de vorm weer en ECLLIPSEMODE(CORNER); geeft weer vanaf welk punt de vorm moet starten.

Assignment 2

De tweede opdracht werd er gevraagd om zoveel mogelijk vormen in het scherm te zetten. Hierbij is het belangrijk dat deze niet te groot zijn en of buiten het scherm vallen. In de handleiding stonden verschillende termen die we konden gebruiken. Omdat het alleen om de vormen ging heb ik ervoor gekozen deze geen kleur te geven. Wanneer je bijvoorbeeld naar rect kijkt in de code kan je zien dat hier vier getallen staan. Deze getallen staan voor de positie van het object, zo geven de eerste twee cijfers de X en Y as aan. Op de afbeelding is ook te zien dat de ruimte aan de zijkanten van de Rect minder is in vergelijking met de bovenkant.

Assignment 3

Eerst leerden we hoe we figuren konden maken in processing, tot slot moesten we bij deze opdracht iets maken met een figuur waarbij deze de iets cools zou doen. In het begin kwam ik er niet echt uit en heb ik daarom ook wat hulp gevraagd aan Chris. Uiteindelijk is dit het resultaat geworden. Wanneer deze code in je processing staat volgt deze de beweging van de muis. Deze opdracht was in voorbereiding op de opdrachten die volgden. Zo volgt deze de beweging van de muis.

Assignment CV1

Bij deze opdracht werd er voor het eerst gebruik gemaakt van de webcam. In de voorgaande opdrachten hebben we geleerd hoe we vormen kunnen maken in dit programma en middels de beweging van de muis de vormen kunnen laten meebewegen. Voorafgaand aan deze opdrach hebben ik een libary gedownload waar een voorbeeld in zat. In combinatie met de studiehandleiding heb ik vervolgends de code in he voorbeeld aangepast.

Assignment CV2

Bij deze opdracht moesten we zo onherkenbaar mogelijk in beeld. Hierbij kreeg ik door de code een zwarte balk voor mijjn ogen. Door _FRONTALFACE en ecllipseMode(CORNER); zorgt dit ervoor dat het balkje in het midden van het gezicht wordt geplaatst.

Assignment CV3

De allerlaatste opdracht werd gevraagd of we met de kennis die we in de workshop hebben opgedaan iets cools wilde maken of laten zien. Ik heb ervoor gekozen om de heart eyes te maken. De code PImage img; zorgt ervoor dat de image wordt aangeroepen. Vervolgens wordt door de regel 15 de imagae geladen. Hierna heb ik de code opgeslagen en deze gerunt, ik miste nog iets in mijn void draw. Dit heb iklater aangevuld en hierna deed de code het.
